# MTS-True-Tech-Hack
LocalScript: локальная агентская система для генерации Lua-кода

## 📂 Структура проекта

```text
.
├── sandbox/              # Всё, что касается запуска Lua (Песочница)
├── src/                  # Основной исходный код проекта (Python)
│   ├── api/              # Бэкенд (FastAPI)
│   ├── agents/           # Логика AI-агента (LangGraph / LangChain)
├── tests/                # Наборы задач (Eval) для проверки ИИ
└── docs/                 # Схемы архитектуры, инструкции, промпты
