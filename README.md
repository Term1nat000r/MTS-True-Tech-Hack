# MTS-True-Tech-Hack
LocalScript: локальная агентская система для генерации Lua-кода

## 📂 Структура проекта
* `src/agents` — логика AI-агента и LangGraph.
* `src/rag` — база знаний и поиск по документации.
* `src/api` — бэкенд на FastAPI.
* `sandbox` — Docker-контейнер для исполнения Lua.
* `tests` — наборы задач для проверки (Eval).
